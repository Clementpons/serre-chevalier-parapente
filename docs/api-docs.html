API Documentation

Serre Chevalier Parapente — Backoffice REST API (Hono + Next.js 15)

Base URL: /api Format: JSON Date: 2026-03-16
Authentification & Middlewares

Toutes les réponses suivent le format standard : { success: boolean, message?: string, data?: T }

aucune auth x-api-key (public) x-api-key + x-session-id (panier) session ou x-api-key session (connecté) moniteur ou admin admin uniquement 
// Headers pour le front public
x-api-key: <NEXT_PUBLIC_API_KEY>
x-session-id: <UUID v4 stocké localStorage, TTL 24h>

// Headers pour le backoffice (better-auth cookie)
Cookie: <better-auth session cookie>

{ success: true, data: { items: CartItem[], total: number }}
POST /add panier ▼

Body

{ type: "STAGE"|"BAPTEME"|"GIFT_VOUCHER",
  itemId?: string,           // stageId ou baptemeId
  giftVoucherCode?: string,  // pour utiliser un bon
  giftVoucherAmount?: number,// pour acheter un bon
  participantData: {
    firstName, lastName, email, phone,
    weight, height, birthDate?,
    // pour BAPTEME
    selectedCategory?, hasVideo?,
    // pour STAGE
    selectedStageType?,
    // pour GIFT_VOUCHER
    voucherProductType, buyerName, buyerEmail,
    recipientName?, recipientEmail?,
    notifyRecipient: boolean
  },
  quantity: number }

Réponse

{ success: true, data: CartItem }
